# Dotfiles (Stef Schenkelaars)
My OS X dotfiles.

## How to install
:warning: Note that this will overwrite your existing dotfiles.

```bash
$ bash -c "$(curl -fsSL raw.github.com/stefschenkelaars/dotfiles/master/bin/install.sh)"
```
